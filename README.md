# Umpire-Decision-System
This project aims to assist cricket umpires by automatically classifying match events from images using Deep Learning.
The system analyzes cricket frames and predicts one of the following umpire decisions:
Six
Wide
No Ball
Out
No Action
It is built using Convolutional Neural Networks (CNN) and trained on a labeled image dataset.
